# Birthday Reminder Bot
A simple Telegram bot that helps you remember birthdays! Store, list, and delete birthdays with easy commands.

## Features
- Add birthdays with /add name YYYY-MM-DD

- List all birthdays with /list

- Delete birthdays with /delete name

- Simple JSON storage

- Error handling

## Commands
- `/start` - Welcome message and instructions

- `/help` - Show available commands

- `/add <name> <YYYY-MM-DD>` - Add a birthday

- `/list` - Show all saved birthdays

- `/delete <name>` - Delete a birthday

## Setup
- Create a Telegram Bot:

- Message @BotFather on Telegram

- Use /newbot command

- Get your bot token

- Install Dependencies:

```bash
pip install python-telegram-bot
```
- Configure the Bot: Replace "MY TOKEN" with your actual bot token

- Run the script

- Start Using:

- Find your bot on Telegram

- Send `/start` to begin

## Example Usage
`text`
`/add Hossein 2001-05-15`
`/list`
`/delete Hossein`
